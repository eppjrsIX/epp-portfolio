Personal Crypto Tracker

"Cryptocurrency is a tool for financial empowerment. It has the potential to level the playing field and give power back to the people." - Andreas Antonopoulos

Description

The Personal Crypto Portfolio Tracker Dashboard provides a real-time, comprehensive overview of your cryptocurrency investments. Leveraging data scraped from the CoinMarketCap API, this dashboard delivers up-to-date insights into your portfolio's performance, helping you make informed investment decisions.

Key Features

  • Total Portfolio Value
  • Total Profit / Loss
  • Overall ROI (%)
  • Top Performing Coin (by % Value Change)
  • Lowest Performing Coin (by % Value Change)
  • Running Total Balance of Selected Portfolio Coins
  • Profit and Loss Breakdown by Selected Portfolio Coins
  • Net Cost vs. Current Value for Selected Portfolio Coins
  • ROI (%) for Selected Portfolio Coins
  • Current Value of Selected Portfolio Coins
  • Interactive Ticker Symbol Selector
